# Ecologies of Representation

Ecologies of Representation is an independent para-academic publishing project. 

This site is being built slowly and publicly, as a form of creative expression, a personal learning trajectory, and most importantly, a small piece of resistance.

Over time, it will host an array of fragments, notes, essays, stories, and whatever else feels relevant to the moment. Through experimentation, I hope to foster both new understandings of and relations with the ecologies of which we all are part, and the knowledge(s) they produce.

The website itself will live at ecologiesofrepresentation.org
